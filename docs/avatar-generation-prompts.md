# Avatar generation prompts

## v1 - Custom couple avatar style calibration

Use the uploaded photo as identity reference for a HomeSync premium couple avatar.

Create a transparent-background premium sticker avatar for a warm household/couple mobile app. Preserve the two main people from the photo: same couple, same approximate ages, same hair colors, same facial vibe, same closeness. Translate them into the exact same visual family as the existing HomeSync premium avatars: cute polished 3D storybook illustration, soft watercolor-like shading, warm cream/peach/sage/honey palette, rounded friendly faces, detailed soft hair, gentle oval eyes with highlights, small noses, warm cheeks, and subtle closed-mouth or barely-open smiles. Avoid toothy caricature smiles, realistic portrait anatomy, sharp cheekbones, tourist caricature style, emoji/memoji style, flat vector style, anime, or photorealism.

Composition: both faces clearly visible and balanced, upper torso/bust only, affectionate and close together, centered, full hair and shoulders visible, no cropping, no circle, no frame, no text, no logo. Use only a tiny hint of the original photo context if useful, but do not copy the pool/swimwear literally if it makes the avatar feel less HomeSync. Prefer cozy casual app-friendly clothing in cream, peach, sage, or warm neutrals. The final character pair should fill most of the square canvas while leaving a small clean transparent margin.
